What is a Vendor?
A vendor represents an AI/LLM provider your organization uses (e.g. OpenAI, Anthropic, Google). Register vendors to track which providers are approved, their data residency policies, and certifications for governance and compliance purposes.
Agent ROI Baseline
Define the manual effort this agent replaces. This baseline is used to calculate ROI by comparing the cost of manual work against the agent's LLM costs. Set how long the task takes manually, the hourly rate, and how often the agent runs.
How ROI is calculated
Labor Saved = Manual Hours × Hourly Rate × Avg Agent Executions
Net ROI = Labor Saved − Actual Agent LLM Cost (from runs)

Example: 1h manual work at $50/h, done 10 times = $500 manual cost. If the agent spent $5 in LLM calls, net savings = $495.
